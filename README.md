# Camouflage

Generative camouflage pattern art built with [p5.js](https://p5js.org/) with vectors with subtle foreground to background differentiation.

## Usage

Open `index.html` in a browser.

- **Click** to regenerate with new colors
- **Press S** to save as PNG

## How it works

Draws random quadrilaterals on a grid with near-identical foreground and background colors, creating a subtle camouflage effect. Each click randomizes from one of six color palettes with a small brightness shift between background and shapes.
